A pseudo-classe :nth-child() faz a correspondência a elementos com base em sua posição entre um grupo de irmãos.

Com ela nós podemos selecionar elementos HTML que existem dentro de outros.

Exemplo: “CSS, selecione o primeiro elemento dentro da div cuja class é cubo, e pinte de amarelo”.

cubo_1:first-child{background-color: yellow; color:black }
